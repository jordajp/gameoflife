gameoflife

A typescript / jquery version of the "Game of life" cellular automaton. More on wikipedia

Dependencies

You'll need:

  • jquery
  • a decent web browser
  • typescript if you whish to modify the code.

Installation

Install a jquery file somewhere, and adapt the url in gameoflife.html.

Run it

Open gameoflife.html with your favorite browser.

You can click on the array to add « live cells » at any time before or during the run.
